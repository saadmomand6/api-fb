Steps to Connect Firebase to flutter app=>

install firebase cli (you can do trough node also)
then after node installation add pub/chache/bin and roamng/npm to path in environment varibles
then open your project throug cmd and run "dart pub global activate flutterfire_cli" 
firebase logout if already login
firebase login
after then run flutterfire configure
then select project or create new
then select platform
then add package name what you want
